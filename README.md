# Game Listings

** Specs 
* As a user:
    - I can create a game
    - I can get all games
    - I can read one game
    - I can destroy one game
    - I should be able to export one game to a text file
    - I should be able to get more info on the location (longitude, latitude)
    
A game should have:
- GameID
- GameName
- UserName
- PhoneNumber
- Price
- PostCode
- Longitude, Latitude
